# PitchPerfect AI

Single-page React training simulator for an AI Advertising Platform sales onboarding program.

## Run locally

```bash
pnpm install
pnpm run dev
```

Open the local URL Vite prints in the terminal.

## Build

```bash
pnpm run build
```

## Vercel

- Framework preset: Vite
- Build command: `pnpm run build`
- Output directory: `dist`

Do not open the root `index.html` directly from the file system. This is a Vite React app, so it needs the dev server locally or Vercel's build step in production.
